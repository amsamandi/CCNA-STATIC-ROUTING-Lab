## Overview

This lab demonstrates static routing configuration between multiple routers using Cisco Packet Tracer.

## Skills demonstrated

* Static route configuration
* Network connectivity verification
* IP addressing
* Router configuration
* Ping and troubleshooting

## Topology

Router R1 connected to Router R2 and R3

Networks:

* 192.168.1.0/24
* 192.168.2.0/24
* 192.168.3.0/24

## Configuration example

Example static route:

ip route 192.168.3.0 255.255.255.0 10.0.0.2

## Verification

Commands used:

show ip route
ping
tracert
